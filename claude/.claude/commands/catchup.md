# Catchup Command

Read the HANDOFF.md file and prepare to continue work from the previous session.

## Instructions

1. **Read HANDOFF.md**: Load the handoff document from the project root
2. **Understand Context**: Parse and understand:
   - What was being worked on
   - Current branch and its purpose
   - Work in progress status
   - Any blockers or issues
   - Planned next steps

3. **Verify Current State**:
   - Check git status to confirm current branch
   - Review any uncommitted changes
   - Verify build/test status if relevant

4. **Summarize for User**: Present a clear summary of:
   - Where we left off
   - What needs to be done next
   - Any outstanding issues to address

5. **Ask for Direction**: Confirm with the user which task to resume or if priorities have changed

## Goal

Seamlessly resume work from the previous session with full context, minimizing ramp-up time.
